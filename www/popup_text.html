Community-Aquired Pneumonia Severity Scores

A interactive tool for the calculation of a range of CAP severity scores from user-uploaded data.

This app calculates 6 different pneumonia severity scores for one or multiple patients. The functions used in this appliation are available as an R package here: https://github.com/maciejrosolowski/progressdatenbankderivate

 Available CAP severity scores:

  • PSI
  • SIRS
  • Quick SOFA
  • Halm
  • SCAP
  • Smart COP

If you need any help click the ? in the header.
